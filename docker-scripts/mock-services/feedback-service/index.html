Feedback Service Mock API

This is a mock Feedback Service for development purposes.

Available Endpoints:

GET /health

Returns the health status of the service.

GET /api/v1/feedback-submissions/available

Returns a list of available feedback submissions for the authenticated user.

POST /api/v1/feedback-submissions/submit

Submits feedback responses.

GET /api/v1/projects

Returns a list of projects.

GET /api/v1/questions

Returns a list of questions.
